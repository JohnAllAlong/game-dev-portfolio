BlockHead

Description

BlockHead is an Unreal Engine project developed for a Winter 2025 course focusing on using Blueprints for gameplay logic. The game includes custom game mechanics, character controls, UI, and interactive elements within the environment.

Goals/Features: Learn and apply Unreal Engine's Blueprint visual scripting system. Implement core gameplay features as specified by the course requirements. Understand UE project structure and asset management.

Challenges/Learnings: Debugging complex Blueprint graphs, optimizing performance, integrating different UE systems (like UI, animation, physics), and managing project scope within course deadlines.
